## Quick Priority Order

If short on time, focus in this order:

1. **C# async/await + Task** — almost always tested
2. **LINQ** — bread and butter for .NET devs
3. **ASP.NET Core DI + middleware pipeline** — senior-level expectations
4. **Angular RxJS operators** — heavily tested for Angular devs
5. **Angular reactive forms** — common coding challenge
6. **SOLID + design patterns** — multiple choice staple
7. **EF Core queries + migrations**
8. **Angular routing guards**
